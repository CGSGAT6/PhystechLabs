Как использовать:

0. установить в python пакет numpy, можно через терминал, можно в виртуальном окружении, если запускаешь например через PyCharm
1. Создать файл, например aaa.txt
2. Заполнить его по схеме:
 + 1-я строка: True если функция должная пересекать начало координат, иначе False
 + 2-я строка: N - количество точек, $N>2$
 + N следующих строк: абсциссы точек, десятичные дроби через точку
 + N следующих строк: ординаты точек, соответствующие абсциссам
3. Запустить файл mnk.py и по желанию через пробел ввести название входного и выходного файлов, стандартные значения in.txt и out.txt
например можно ввести ничего и просто нажать Enter, ввести aaa.txt, или ввести aaa.txt bbb.txt
4. Если введённые данные корректны, то в выходном файле появится результат обработки

Пример файла in.txt:
true
3
1
2
3
-1
-2.1
-2.9
